Wybaczcie opóźnienie, ale przygotowania do webinaru zjadły cały mój czas.

Mając podany wyraz, znajdź palindrom, który zostanie utworzony poprzez dodanie jak najmniejszej liczby znaków jak to możliwe w dowolne miejsce w tym słowie. String po dodaniu liter nie musi tworzyć poprawnego słowa.
Jeśli można stworzyć więcej niż jeden palindrom używając tej samej najmniejszej liczby znaków, zwróć ten który jest pierwszy pod względem alfabetycznym.

Na przykład, podając słowo "rota", program powinien zwrócić string "atorota, ponieważ tworzymy palindrom dodając jedynie 3 litery. 
Można w tym przykładzie stworzyć inne palindromy dodając 3 litery np. "rotator", ale "atorota" jest wcześniej pod względem alfabetycznym.

Innym przykładem jest słowo "google", dla którego program powinien zwrócić string "elgoogle".
